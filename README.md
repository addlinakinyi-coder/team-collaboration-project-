# team-collaboration-project-
A repository to practice forking, branching and pull request 

## 🤝 Collaboration Goal
The goal of this task is to practice **Forking**, **Branching**, and **Pull Requests**.

## 🚀 How to Contribute
1. **Fork** this repository to your account.
2. Create a new **Branch** named `your-name-update`.
3. Add your name to the contributors list below.
4. Open a **Pull Request**.
